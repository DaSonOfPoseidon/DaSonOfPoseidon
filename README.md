### About Me

- Mizzou undergrad who builds software to fix real operational problems — started when manual scheduling in the fiber installation industry was costing time and money
- Designed and self-host a home server running Docker-deployed services with Cloudflare Tunnels, automated repo syncing, speed monitoring, and a custom tunnel manager that auto-discovers containers and handles URL creation
- Backend-focused — Python, FastAPI, async task queues, data pipelines — with enough frontend to ship complete solutions
- Motivated by finding inefficiencies, breaking them down, and engineering reliable systems around them
